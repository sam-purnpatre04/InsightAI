# InsightAI

InsightAI is an AI-powered Business Intelligence platform designed to help retail and e-commerce businesses transform raw data into actionable insights. The platform automates data cleaning, exploratory data analysis (EDA), machine learning, explainable AI, and interactive business reporting in a single application.

---

## Project Objective

The goal of InsightAI is to simplify business data analysis for non-technical users by providing an end-to-end analytics platform. Users can upload datasets, explore trends, generate predictions, and receive AI-generated business insights without requiring advanced data science knowledge.

---

## Key Features

- Upload CSV and Excel datasets
- Automatic data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Interactive visualizations
- Machine Learning model training
- Explainable AI using SHAP
- AI-powered business insights
- PostgreSQL database integration
- Power BI dashboards
- Report generation
